# Arquitectura de Vantage

## Descripción General

Vantage es un proyecto full stack con arquitectura de monorepo.

## Componentes

### Backend
- Express.js para el servidor HTTP
- TypeScript para type safety
- PostgreSQL para persistencia de datos
- Estructura en capas: controllers, services, repositories

### Frontend
- React para UI
- TypeScript para type safety
- Comunicación con backend a través de APIs REST

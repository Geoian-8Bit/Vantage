# Vantage — Control de gastos personales

Aplicación de escritorio para Windows que te permite llevar un registro detallado de tus ingresos y gastos personales, ver estadísticas y analizar tus hábitos financieros.

## Características

- **Panel de inicio** — balance total, gastos del mes, categoría principal y tendencia de los últimos 6 meses
- **Movimientos** — registro de ingresos y gastos con descripción, categoría, fecha y notas; filtros por tipo, período y categoría; búsqueda por texto; exportación a Excel
- **Estadísticas** — gráficos de barras, circular por categorías, evolución del balance y mapa de calor por día de la semana; exportación a PDF
- **Calendario** — vista mensual con totales diarios de ingresos y gastos
- **Transacciones recurrentes** — plantillas semanales, mensuales, trimestrales o anuales que se crean automáticamente al abrir la app
- **Importación** — importa movimientos desde archivos Excel (.xlsx / .xls) o bases de datos Microsoft Access (.mdb / .accdb) con un asistente paso a paso
- **Copia de seguridad** — guarda y restaura toda la base de datos en un archivo .db
- **Temas** — varios esquemas de color para personalizar la interfaz

## Instalación

1. Descarga `Vantage Setup x.x.x.exe`
2. Ejecútalo y sigue el asistente (instalación de un solo clic)
3. Abre Vantage desde el escritorio o el menú de inicio

> Al instalar una versión nueva sobre una ya existente, tus datos se conservan íntegramente.

## Tecnología

- **Electron 41** + **React 19** + **TypeScript**
- **SQLite** (sql.js) — base de datos local, sin servidores ni conexión a internet
- **Tailwind CSS** · **Recharts** · **PDFKit** · **xlsx**

# Vantage — Personal expense tracker

A Windows desktop application to keep a detailed record of your personal income and expenses, view statistics, and analyze your spending habits.

## Features

- **Dashboard** — total balance, monthly expenses, top spending category, and 6-month trend chart
- **Transactions** — log income and expenses with description, category, date, and notes; filter by type, period, and category; text search; Excel export
- **Analytics** — bar charts, category pie chart, balance evolution, and weekday spending heatmap; PDF export
- **Calendar** — monthly view with daily income and expense totals
- **Recurring transactions** — weekly, monthly, quarterly, or annual templates that are automatically created when the app opens
- **Import wizard** — import transactions from Excel files (.xlsx / .xls) or Microsoft Access databases (.mdb / .accdb)
- **Backup & restore** — save and restore the entire database to/from a .db file
- **Themes** — multiple color schemes to customize the interface

## Installation

1. Download `Vantage Setup x.x.x.exe`
2. Run it and follow the one-click installer
3. Open Vantage from your desktop or start menu

> Installing a new version over an existing one preserves all your data.

## Tech stack

- **Electron 41** + **React 19** + **TypeScript**
- **SQLite** (sql.js) — local database, no servers or internet connection required
- **Tailwind CSS** · **Recharts** · **PDFKit** · **xlsx**
